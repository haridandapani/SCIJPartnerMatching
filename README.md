# SCIJPartnerMatching
CS For Social Change Project for Matching Students with Attorneys

To make the virtual env: 
- cd to /backend
- Windows: 
    - python -m venv venv
    - pip install -r requirements.txt
    - venv\Scripts\activate
- Mac/Linux:
    - python3 -m venv venv
    - pip3 install -r requirements.txt
    - source venv/bin/activate
